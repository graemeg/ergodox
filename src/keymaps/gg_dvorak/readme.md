I was a long time TypeMatrix 2030 user with the Dvorak layout. This layout
takes some cues from the TM2030. I've also made some tweaks for usage with
the Object Pascal IDEs I work with daily (F-keys, Ctrl, Alt and arrow keys
are used very often).

I'm currently experimenting with two different symbols/numpad layers too
see which one works best for me.

I'm also evaluating the BEAKL-9 layout to see if that is any better than
my trusty Dvorak layout.

Graeme Geldenhuys.
